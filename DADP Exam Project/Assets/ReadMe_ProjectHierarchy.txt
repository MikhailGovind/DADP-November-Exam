Hierarchy created by R-D (1612948@students.wits.ac.za) on 04/11/2022.


1. Managers - collection of all managers used in Scene

2. Setup - collection of all scene basic essentials, e.g. cameras, eventsystem, etc.

3. Environment - all the elements that make up the environment of the scene, e.g. player, enemy units, tiles/grid

4. Canvases - all canvases used in scene

5. System - all the persistent systems used and passed from scene to scene (i.e.DontDestroyOnLoad)
